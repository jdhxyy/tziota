# TZ-IOT发布Python SDK：tziota
本文博客链接:http://blog.csdn.net/jdh99,作者:jdh,转载请注明.

## **简介**
在[《物联网平台TZ-IOT发布透传云内测服务：V1.0》](https://blog.csdn.net/jdh99/article/details/90638387)中已经上线TZ-IOT的内测服务，本文发布Python SDK。

### 用途：
不用租用公共服务器，任意选择一台可联网的PC或者树莓派这样的嵌入式设备，即可当作服务器。与同样基于TZ-IOT运行的节点通信。

## **安装**
pip install tziota
